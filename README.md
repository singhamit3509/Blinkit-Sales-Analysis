Blinkit Sales Dashboard (Power BI Project)

Overview
This project presents an interactive Sales Analytics Dashboard for Blinkit (India's last-minute delivery app). 
The goal is to analyze sales trends, outlet performance, and product insights to support business decision-making.

Features

Sales Overview
Total Sales: $1.20M
Average Sales: $141
Total Items Sold: 8523
Average Rating: 3.9

Outlet Analysis

Sales comparison by Outlet Type (Supermarket, Grocery Store)
Outlet Tier performance (Tier 1, Tier 2, Tier 3)
Outlet Size contribution (Small, Medium, High)

Product Insights

Category-wise sales (Fruits, Snacks, Dairy, etc.)
Fat content distribution (Low Fat vs Regular)
Item visibility impact on sales

Trend Analysis
Year-wise outlet establishment & sales trends

Interactive Filters

Outlet Location Type
Outlet Size
Outlet Type
